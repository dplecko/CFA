Causal Fairness Analysis - Software Tools

This page contains a sequence of vignettes associated with the Causal Fairness Analysis paper (Plecko and Bareinboim 2022). In particular, here you can find all the code used to reproduce the results of the paper, formatted in a vignette-style fashion. There are also additional examples of analyses, which are not presented in the manuscript. In particular, you can find the following:

A: Vignettes directly reproducing the content of the manuscript

  1. Census dataset analysis from Example 5.1 of (Plecko and Bareinboim 2022, sec. 5.1),

  2. College Admissions Bias Quantification over Time from Example 5.2 of (Plecko and Bareinboim 2022, sec. 5.1),

  3. Analysis of fair predictions on the COMPAS dataset from Example 5.3 of (Plecko and Bareinboim 2022, sec. 5.2),

  4. COMPAS dataset analysis from Example 6.1 of (Plecko and Bareinboim 2022, sec. 6),

  5. Conditional independence tests on COMPAS & Census datasets from Figure 4.7 of (Plecko and Bareinboim 2022, sec. 4.5),

  6. Outcome Control Cancer Surgery Example from (Plecko and Bareinboim 2022, sec. 5.3.3).

B: Vignettes not directly reproducing the content of the manuscript:

  1. TV & EO analysis on COMPAS dataset,

  2. Fair predictions using Fair Data Adaptation.

To learn more about Causal Fairness Analysis, visit https://fairness.causalai.net/ where you can access a 2-hour Tutorial on the topic, which was presented in the International Conference on Machine Learning (ICML) 2022.

References

Plecko, Drago, and Elias Bareinboim. 2022. “Causal Fairness Analysis.” arXiv Preprint arXiv:2207.11385.
